# vk-raid-botnet

# Установка

`git clone https://github.com/Madara225/vk-raid-botnet`

`cd vk-raid-botnet`

`pip3 install -r requirements.txt`

# Добавление аккаунтов

`cd sessions`

`python3 add_account.py`

1 - добавляет аккаунты
2 - проверяет аккаунты

# Конфиг

`cd settings`

`nano config.py`

# Запуск

`python3 main.py`
